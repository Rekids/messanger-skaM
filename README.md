# messanger-skaM
P2P-мессенджер для локальных сетей (LAN/Radmin/ZeroTier). Никаких серверов — сообщения идут напрямую между компьютерами. Нельзя заблокировать, потому что нечего блокировать. Никакой регистрации, никакой слежки. Просто укажи IP друга и общайся. Работает там, где официальные мессенджеры уже нет.
